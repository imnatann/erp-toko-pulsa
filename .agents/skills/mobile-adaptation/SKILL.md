# Skill: Mobile Adaptation

## Tujuan

Mengadaptasi web yang sudah stabil menjadi pengalaman yang nyaman di tablet dan HP, lalu menyiapkan handoff ke app mobile bila perlu.

## Prinsip

- tablet adalah target pertama setelah desktop
- mobile fokus ke tugas cepat: cek pending, validasi, lihat ringkasan, input sederhana
- jangan paksa seluruh pengalaman admin desktop ke HP bila tidak masuk akal

## Prioritas adaptasi

- queue transaksi pending
- detail validasi
- dashboard ringkas
- form transaksi digital sederhana
- POS tablet layout

## Handoff ke native mobile bila nanti dibutuhkan

- identifikasi fitur yang butuh akses perangkat lebih dalam
- pertahankan API/domain model yang konsisten dengan web

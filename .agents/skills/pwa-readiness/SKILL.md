# Skill: PWA Readiness

## Tujuan

Menyiapkan web agar nanti bisa dipasang di tablet/HP dan punya pengalaman aplikasi yang lebih baik.

## Dasar riset

- PWA harus cepat, installable, usable lintas browser, responsive, dan punya offline behavior yang masuk akal.

## Fokus untuk project ini

- manifest dasar
- icon set
- install prompt yang masuk akal
- offline fallback page
- caching hanya untuk aset yang aman

## Catatan penting

- transaksi yang butuh kepastian real-time tidak boleh diam-diam dianggap sukses saat offline
- offline mode lebih cocok untuk melihat queue, draft lokal, atau fallback page, bukan finalisasi finansial

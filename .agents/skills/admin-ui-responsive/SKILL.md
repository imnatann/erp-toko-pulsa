# Skill: Admin UI Responsive

## Tujuan

Memastikan web admin/POS tetap nyaman di desktop, tablet, dan HP.

## Prinsip dari riset

- responsive ke semua layar
- jangan ada fitur penting yang hilang di viewport kecil
- prioritaskan tindakan utama dan konten penting

## Aturan UI

- tabel besar harus punya fallback card/stack di layar kecil
- filter sekunder boleh collapse
- tombol aksi penting harus mudah dijangkau
- validasi manual transaksi harus bisa selesai dari tablet/HP
- hindari modal terlalu kompleks di layar kecil

## Fokus halaman penting

- daftar pending transaction
- detail validasi
- POS barang fisik
- dashboard owner
